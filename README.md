# Chloe Theme

This is my simple oh my zsh theme based on robbyrussell default theme.

<p>To install: </p>


<code>
  $ git clone https://github.com/BrunoMoreno/chloe-theme.git

  $ cd chloe-theme-master && cp chloe.zsh-theme ~/.oh-my-zsh/themes/
</code>

Change in .zshrc:
ZSH_THEME = "chloe"

<code>
  $ source .zshrc
</code>



Enjoy It!
